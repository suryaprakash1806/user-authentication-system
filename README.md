# PHP MySQL CRUD User Management System

# Description

A simple PHP and MySQL user authentication and CRUD application developed using XAMPP. The project allows users to register, login, view records, update information, and delete users.

## Features

* User Registration
* User Login
* Session Management
* Create User
* Read Users
* Update User
* Delete User
* MySQL Database Integration

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP

## Project Structure

* register.php
* login.php
* dashboard.php
* logout.php
* manage_users.php
* edit_user.php
* delete_user.php
* db.php
